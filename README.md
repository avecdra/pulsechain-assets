## How to add a PulseChain token

### Requirements
- Token must be deployed on PulseChain (Chain ID: 369)
- Project must be legitimate with available information
- Token should have non-minimal circulation
- High-quality logo (PNG format, minimum 256x256px)

### Simple Process
1. **Fork this repository**
2. **Create token folder**: `blockchains/pulsechain/assets/0xYOUR_TOKEN_ADDRESS/`
3. **Add required files**:
   - `logo.png` (256x256px minimum, PNG format)
   - `info.json` with token metadata
4. **Submit a pull request**

We will review and merge legitimate token additions. For detailed `info.json` format, see existing tokens in the repository.
